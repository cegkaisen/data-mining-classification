# Codex Proje Notlari

Bu klasor AI destekli muhendislik (AI-assisted engineering) icin yerel notlari tutar.

Baslangic dosyalari:

- `AGENTS.md`: ajan davranisi ve proje kurallari.
- `docs/ai/context.md`: mevcut proje anlayisi.
- `docs/ai/project-brief.md`: assignment ozeti ve teslim gereksinimleri.
- `docs/ai/workflow.md`: varsayilan calisma dongusu.
- `docs/ai/decision-log.md`: kalici kararlar.
- `docs/ai/task-template.md`: cok adimli gorevler icin sablon.
